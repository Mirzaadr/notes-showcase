# Notes Showcase - Documentation

This directory contains the main documentation pages for the Notes Showcase app.

## Purpose
This app is built to showcase a wide variety of notes, ranging from beginner "getting started" guides to advanced "simple tips" for each topic. 

## Folder Structure
- `devops/`: Everything related to DevOps.
- `dotnet/`: Everything related to .NET programming.

Please navigate to the specific folders and read the `welcome.md` files to understand the specific scope of each topic!
